Put all the .py modules, .db and csv files in the same project directory.
You may need to download some external packages, such as geopy, requests etc.
Learn how to install packages by searching "pip" online.

General steps:
1. Install pip
2. Run pip from command line mode:
   pip install [package_name]


Start the program by running pokemonGo_main.py




Database tables are already created by create_table.py. 
If you need to make changes to the database schema, you can modify this file and run it.
